授業概要
／ 		コンピュータグラフィックス(以下「CG」)、特に3次元(以下「3D」)を扱う高度な手法を体系的に説明する。3DCGには用途によって様々な技法があるが、それぞれの特性を基礎理論から理解させ、実践的な場面での効果的な利用法の修得を指導する。前半は、線形代数を基礎とする3DCGの理論的大系について学ぶ。後半は、シェーダーによるリアルタイム3DCGプログラミングの技法について学ぶ。

This course systematically explains advanced techniques for computer graphics (hereafter “CG”), particularly three-dimensional (hereafter “3D”) graphics. There are various techniques for 3DCG depending on the application, but this course will help you understand the characteristics of each technique from the basic theory, and teach you how to use them effectively in practical situations. In the first half of the course, you will learn about the theoretical system of 3DCG based on linear algebra. In the second half, you will learn about techniques for real-time 3DCG programming using shaders.
到達目標
／ 		リアルタイム3DCG分野の論文を理解でき、実際に実装する能力を習得することを目標とする。

The goal is to be able to understand papers in the field of real-time 3DCG and to acquire the ability to actually implement them.
ラーニングアウトカムズ（学修到達目標）
／ 		実学に基づく専門能力、論理的な思考力、問題解決力

授業方法
／ 		基本的には講義形式で行い、配付資料と板書を用いて解説を行う。途中、記述による問題を課すことがある。ノートPCは必携とする。

The lecture will be given in a lecture-style format, with explanations given using handouts and the blackboard. There may be written questions during the lecture. A laptop computer is required.
履修上の注意
／ 		本科目はC#言語を用いて三次元基本形状をプログラムによって表示する方法を既に習得していることを前提とする。これらに関しては、課題を行うのに必要となる。

This course assumes that you have already learned how to display basic three-dimensional shapes using the C# programming language. You will need to do the assignments to learn this.
準備学習
／ 		C#言語による基本的なプログラミング技術を習得済であることが前提となるので、履修を希望する学生は必ず事前に学習しておくこと。各単元は事前準備(60分を目安)を提示するので、事前に実施しておくこと。また、各単元で課題(180分を目安)とするので、毎回提出すること。

Students wishing to take this course must have already mastered basic programming techniques using the C# language. Each unit will be preceded by a preparatory exercise (60 minutes), so please complete this in advance. In addition, each unit will be followed by an assignment (180 minutes), so please submit this each time.
成績評価方法・基準
／ 		毎回の課題に対し100点満点で点数をつけ、合計点が60点以上となることで単位修得となる。また、合計点により成績評価も決定する。各課題に対しては、解答例を示すなどのフィードバックを実施する。

Each assignment is graded out of 100 points, and a total of 60 points or more is required to pass the course. Grades are also determined based on the total score. Feedback is provided for each assignment, including sample answers.

授業計画
／ 		第1回: ベクトルと行列 (線形代数、ベクトル、内積、外積)
第2回: 線形変換 (同次座標、回転変換、転置行列、直交行列、系、合成変換)
第3回: 射影の数学 (射影、正射影、透視射影、視野変換、視錐台)
第4回: シェーダーの基本 (GPU, シェーダー)
第5回: 頂点シェーダー (並列演算、シェーダー変数)
第6回: フラグメントシェーダー (線形補間、画像処理)
第7回: シェーダーの応用 (エッジ抽出、デプスバッファ)

Lesson 1: Vectors and Matrices (Linear algebra, vectors, inner product, outer product)
Lesson 2: Linear Transformations (Homogeneous coordinates, rotation transformation, transpose matrix, orthogonal matrix, system, composition transformation)
Lesson 3: Projection Mathematics (Projection, orthographic projection, perspective projection, view transformation, viewing frustum)
Lesson 4: Shader Basics (GPU, shader)
Lesson 5: Vertex Shaders (parallel processing, shader variables)
Lesson 6: Fragment Shaders (linear interpolation, image processing)
Lesson 7: Shader Applications (edge extraction, depth buffer)
